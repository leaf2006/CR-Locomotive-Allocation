# CR-Locomotive-Allocation

**这是一个自动化从多种信息源获取中国铁路各机车与动车组列车配属的项目**

还未完工，正在开发中...

## 简介
本项目计划实现抓取国内绝大部分正在运用的机车、动车组车号与配属，并按照车型与配属各自分类，并制作成一个网站方便查阅。其中车号、车型、配属、生产厂家、备注为本地存储，图片url则使用下关站。

## 数据源:
- [下关站](http://www.xiaguanzhan.com)
- [emu.passearch.info](https://emu.passearch.info/)

## 技术栈

- **前端（网页显示）：** TypeScript + Vite + Vue + Native UI
- **后端（数据获取）：** Python + Github Action

## 免责声明

本项目仅供学习交流使用，获取的数据全部开源，不得用于商用。

本人也是下关站的忠诚用户之一，下关站在收集国铁机车与动车组户口的贡献是开创性的，但是下关站在很多时候会无法访问或访问速度、搜索速度极慢，其网站内部也没有关于路局、机务段配属的分类，因此抓取下关站的数据，创建本项目。
